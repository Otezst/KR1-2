Виды мишеней

Мишень Описание
Используется для статичной стрельбы из пистолетов на дистанции 10-25 метров. Так же можно использовать для стрельбы из огнестрельного оружия ограниченого поражения (травматический пистолет или револьвер) на дистанции 5-7 метров. Высота: 500 мм
Используется для стрельбы из длинноствольного оружия на дистанции от 50 метров. Так же применяется при стрельбе из спортивных пистолетов и револьверов на дистанции 25 метров. Диаметр: 500 мм
Используется для выполнения упражнения МП-5 на 25 метров, а так же для стрельбы из охотничьих нарезных карабинов и гладкоствольных ружей. Высота: 1 600 мм
